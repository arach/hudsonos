# NPM Packages

Published packages for building and managing Hudson apps.

## @hudsonos/sdk

The core SDK for building Hudson apps -- types, hooks, utilities, and components.

```bash
bun add @hudsonos/sdk
```

- [Getting Started](sdk/getting-started.md) -- Installation and first app
- [Building Apps](sdk/building-apps.md) -- Deep dive into app architecture
- [API Reference](sdk/api-reference.md) -- Types, hooks, and function reference
- [Systems](sdk/systems.md) -- Intents, services, and inter-app data piping
- [Utilities](sdk/utilities.md) -- Platform adapters, design tokens, and UI sounds

## @hudsonos/hx

Local service registry and smart router for Hudson.

```bash
bun add -g @hudsonos/hx
```

- [Guide](hx/hx.md) -- Daemon lifecycle, service registration, proxy, API reference
